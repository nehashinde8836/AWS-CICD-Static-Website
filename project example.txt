Solution: Automated CI/CD Pipeline

We build a pipeline that automates everything:

CodeCommit (Code Repository)

All developers push code here.

Acts like GitHub but inside AWS.

CodePipeline (Orchestrator)

Detects when new code is pushed.

Starts the automated build â†’ test â†’ deploy process.

CodeBuild (Automated Build & Test)

Pulls the latest code.

Installs dependencies (npm install / pip install).

Runs tests (npm test / pytest).

Compiles/builds the project.

If tests fail â†’ pipeline stops â†’ developer notified.

Deployment

Option 1: EC2

Deploys backend API or full-stack app to a running EC2 instance.

Option 2: S3

Deploys static frontend (HTML, CSS, JS) to an S3 bucket for a website.

Automatically updates without downtime.

Monitoring & Notifications

Optional: CloudWatch + SNS sends notifications if the pipeline fails.
